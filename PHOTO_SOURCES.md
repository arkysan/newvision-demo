# New Vision Photo Sources

The demo uses local image files only. Vehicle listing photos in `img/` are copied from the New Vision/nvncar inventory URLs that were already referenced by the demo data, plus existing local inventory images from this project.

- `img/v01-tesla.webp` through `img/v22-livan.webp`: existing local New Vision demo inventory assets.
- `img/nvn-v14-jetour-dasheng.webp` through `img/nvn-v35-jetour-shanhai-t2.jpg`: local copies of `nvncar.com/data/watermark/...` listing photos originally referenced by the New Vision demo data.

Do not hotlink remote inventory photos from the page. Add new vehicle photos to `img/`, then point vehicle data at the local file.
